hxScout

A free, cross-platform and open source alternative to Adobe Scout, hxScout is a profiling tool for Flash (and soon Haxe*) applications that displays frame timing, memory usage, and profiling information.

Download

Version: 0.1 Beta

Please note that hxScout is in beta - some features are partially implemented, others are planned, and there are probably bugs. Please submit any feedback (comments, bugs, feature requests) to me via Twitter or github.

Windows Mac OS X Linux

Usage

Profiling with hxScout works just like with Scout. You need to setup a .telemetry.cfg file and compile your SWF with -advanced-teletmetry as documented in Scout's Getting Started Guide.

Features

In the future, this section will compare the feature sets of Adobe Scout and hxScout side-by-side. In version 0.1 beta, hxScout displays timing, profiling and memory allocation data, though it doesn't currently have all the options that Scout does (sorting information, top-down and bottom-up options, etc). hxScout does not support rendering or Stage3D profiling.

The goal is not to replicate every feature of Scout, but to provide a free, cross-platform tool that supports the highest-value profiling functions.

* Haxe Profiling Support

Support for profiling Haxe applications (hxcpp definitely, perhaps neko and other targets) is planned. For now, hxScout functionality will be vetted against the existing FLM telemetry generators (Flash Player and AIR).
